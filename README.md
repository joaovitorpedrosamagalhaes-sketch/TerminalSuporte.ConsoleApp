# TerminalSuporte.ConsoleApp
As heurísticas de usabilidade foram aplicadas no sistema de forma a melhorar a experiência do usuário e reduzir erros.

A Heurística #5 – Prevenção de Erros foi implementada na funcionalidade de formatação de unidade. Antes de executar essa ação crítica, o sistema solicita uma confirmação do usuário (“sim/nao”), evitando que a operação seja realizada acidentalmente.

A Heurística #6 – Reconhecimento em vez de Memorização foi aplicada por meio de um menu de comandos sempre visível na tela. Dessa forma, o usuário não precisa memorizar comandos, podendo apenas reconhecer as opções disponíveis, como “Ping”, “Formatar”, “Help” e “Sair”.

A Heurística #10 – Ajuda e Documentação foi atendida com a criação do comando “help”, que exibe uma explicação simples sobre cada funcionalidade do sistema, permitindo que o usuário tire dúvidas rapidamente sem sair da aplicação.

Assim, o sistema se torna mais intuitivo, seguro e fácil de usar.
